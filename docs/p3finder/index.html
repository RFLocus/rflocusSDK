p3finder

UHF帯RFIDタグが貼付された物品の位置を高速・高精度にサーチするためのソフトウェア開発キットです。 複数タグの水平・垂直位置をスマートフォン画面に高精度でレーダー表示することができます。 電波位相情報を用いることで検索対象タグに近づいているか離れているかを、 従来の電波強度のみを利用する方式に比べ、センチメートル単位の高い精度で判定可能です。

P3finder画面構成

  • ①showTitle: この設定をfalseにすると、「Top View/Side View」のタイトルが隠されます。

  • ②showCount: この設定をfalseにすると、カウントのTextViewが隠されます。

  • ③showStatus: この設定をfalseにすると、ステータスのTextViewが隠されます。

  • ④showRssiBar: この設定をfalseにすると、RSSIバーが隠されます。

  • viewTpe

    • top_side: Default設定。Top ViewとSide Viewが両方表示されます。

    • top: Top Viewしか表示しないです。

    • side: Side Viewしか表示しないです。

Sample

以下のサンプルコードはR3HandyEngineを用いて、タグの読取を行なっています。

class MainActivity : RfidActivity(){
    private val TAG = "P3FindSDK"
    private lateinit var mP3FinderView: P3FinderView

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        //add this before p3finder view layout
        P3FinderSdk.sdkInitialize(
            applicationContext
        ) { result, message ->
            Log.d(
                TAG,
                "onAuthenticated: $result,$message"
            )
        }
        setContentView(R.layout.activity_main)
        mP3FinderView = findViewById(R.id.p3finderView)

    }

    //タグを受信した際に、タグの情報をP3FinderViewに渡します。
    override fun reportTag(tagData: TagData) {
        mP3FinderView.addTagData(tagData.epc, tagData.rssi, tagData.phase, tagData.frequency, tagData.polarization)
    }

    //トリガーが押された際の処理です。
    override fun reportTrigger(pressed: Boolean) {
        doWork(pressed)
    }

    //タグの読み取りを開始/停止する際に、P3FinderViewに計算の開始/停止を通知します。
    override fun reportCommand(cmd: Enum.ReaderCommand,result: Boolean, detail: String?) {
        if (cmd == ReaderCommand.startInventory && result) {
            mP3FinderView.start()
        } else if (cmd == ReaderCommand.stopInventory && result) {
            mP3FinderView.stop()
        }
    }

   //読取開始・停止します。
    fun doWork(pressed: Boolean){
        if(pressed){
            mReader?.startInventory()
        }else{
            mReader?.stopInventory()
        }
    }
    
}
